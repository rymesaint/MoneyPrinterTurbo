# SOUL.md - The Core Spirit of MoneyPrinterTurbo

## Core Vision
Democratize and automate video content creation. Turn any keyword, idea, or prompt into a fully-produced, high-quality, syndicated short video with absolute zero manual editing effort.

## Core Philosophy
1. **Frictionless Automation**:
   - Every tedious step (writing scripts, selecting B-roll, synthesizing speech, timing subtitles, rendering transitions, uploading) must be fully automated.
   - The user inputs a thought; the machine outputs a polished video.
2. **Pragmatic Simplicity**:
   - Avoid over-engineered abstractions. The backend is a pipeline that processes text to script, script to audio/subtitles, searches assets, and renders via MoviePy. Keep it transparent and linear.
3. **Inclusive Integration**:
   - Support multiple LLM backends (OpenAI, Gemini, DeepSeek, Local models), multiple stock providers (Pexels, Pixabay, Coverr), and multiple platforms. The user controls their supply chain.
4. **End-to-End Syndication**:
   - A video sitting on a hard drive is useless. Automation is only complete when the video is published directly to TikTok, YouTube, and Instagram (via cross-posting integrations).
